------ Proje Detayları Hakkında -------

Projemizde veri tabanı değil de dosyalarla çalışma sağladığımız için npm json server kullandık http server). 

npm detayları hakkında: 

-Projenin kurulu olduğu dizine gidip cmdyi açıyoruz,
-Terminalden, npm install -g jspn-server girişini yaptıktan sonra,
-json-server --host (internet bilgilerinizde yazan IPv4 adresinizi yazın, örnek 192.168.1.35:3000),
-Ardından dosya yolunu yazın  (lib/data/students.json)
-user_service dosyasında IP bilgilerimizi giriyoruz.
